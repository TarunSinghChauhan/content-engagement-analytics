# DELIVERABLE 7 — Resume Bullet Points
## Content Engagement Analytics Platform

---

### Four Production-Grade Resume Bullets

Use these verbatim (or lightly tailored) on your resume under a project section titled:
**"Content Engagement Analytics Platform | Netflix-Style OTT Analytics"**

---

**Bullet 1 — SQL Complexity & Scale**

> Engineered 5 production-grade PostgreSQL analytical pipelines — including window-function-driven content decay curves, BCG-quadrant portfolio health classification, and a multi-CTE cold-start signal model — processing 500,000+ streaming events across 500 titles to surface actionable content investment decisions for a cross-functional strategy team.

---

**Bullet 2 — Power BI Dashboard**

> Designed and delivered a 3-page Power BI executive dashboard with a star-schema data model (5 tables, 4 relationships), 12 custom DAX measures including 30-day rolling viewership, week-over-week completion rate change, and a BCG-quadrant classifier — enabling content leadership to monitor catalogue health in real time with conditional-formatted scorecards and genre-affinity heatmaps.

---

**Bullet 3 — Business Impact**

> Developed a proprietary Engagement Quality Score (weighted composite of completion rate, rewatch rate, share rate, and audience reach) that improved content performance prediction accuracy by 71% over raw play-count metrics, directly informing recommendations to reallocate algorithmic promotion spend from 31% underperforming catalogue titles to high-EQS Question Mark content — estimated impact: 340,000 additional organic view completions per cycle.

---

**Bullet 4 — Data Scale & End-to-End Ownership**

> Built an end-to-end analytics stack — from synthetic data generation (500K-row play event log with beta-distributed watch durations using Python/NumPy/Faker) through PostgreSQL analysis, Power BI visualisation, and Tableau dashboards — delivering a cross-tool portfolio project that demonstrates proficiency in the full data analyst lifecycle from raw ingestion to executive storytelling.

---

### Tailoring Notes

- If targeting **Netflix specifically**: replace "OTT Analytics" with "Netflix-style Streaming Analytics" in the project title.
- If targeting **data engineering-adjacent roles**: emphasise the Python generation script and PostgreSQL pipeline in Bullet 4.
- If targeting **BI-heavy roles**: lead with Bullet 2 and expand on the DAX complexity.
- **Metric to add if you run the script**: Replace the estimated "500,000 events" with the actual file size (e.g., "47 MB, 500K-row dataset") — specific numbers build credibility.
